# VIN Decoder Project

Тестове завдання для розробника. Додаток дозволяє отримувати інформацію про автомобіль за його VIN-номером.

## Технології:

- React (Vite)
- API: NHTSA vPIC
- Deployment: Netlify

## Як запустити:

1. `npm install`
2. `npm run dev`
